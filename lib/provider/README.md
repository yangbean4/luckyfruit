多provider通信

获取属性 直接通过init时传入的其他实例获取
调用方法通过 event bus

- LuckyGroup 各种其他奖励相关 如双倍金币、自动合成、合成树时送升级树的概率 等等
- TourismMap 地图系统相关
- MoneyGroup 金币系统相关
- TreeGroup 所有树操作相关 合成、仓库等等
